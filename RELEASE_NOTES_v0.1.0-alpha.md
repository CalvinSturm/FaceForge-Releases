# FaceForge v0.1.0 Alpha

FaceForge is a privacy-first local Windows video tool for face blur, redaction, and consent-based face replacement workflows.

## Highlights

- Blur Faces / Privacy is now the default workflow
- Free tier includes a real visible watermark
- Free exports are limited to 60 seconds
- Face replacement is gated behind upgrade and consent flow
- Windows app icon and packaging now use FaceForge branding
- Release ZIP is packaged for NVIDIA Windows systems

## Known limitations

- Alpha release
- Free 720p cap is not enforced yet
- Watermark is a simple text overlay
- Face replacement requires consent and proper rights
- Some optional model/runtime features may require local setup
- Public alpha ZIP builds may exclude third-party model weights until redistribution rights are confirmed
- Do not upload sensitive videos in public support issues
