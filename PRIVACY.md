# FaceForge Privacy Notes

FaceForge is local-first. The core video processing workflow runs in the
Windows desktop app on your machine.

## Local Processing

- Target videos are processed locally by the desktop app.
- Output files are written to local paths you choose or to app-local output
  locations.
- Model files, logs, cache files, and artifacts may be stored locally by the
  app.

Do not treat this alpha as independently audited privacy software. Review
outputs and local files carefully when working with sensitive footage.

## Support and GitHub Issues

Do not upload private or sensitive videos to GitHub issues.

Do not upload:

- private videos
- source face images
- IDs or documents
- footage involving minors
- workplace, medical, legal, or financial footage
- any media you do not have permission to share publicly

Use a test clip, redacted screenshot, or written description whenever possible.

## Logs and Support Bundles

Logs and support bundles may contain system details, file paths, settings,
hardware information, runtime messages, and error details. Review any log or
support file before sharing it publicly.

If a support file includes private paths, names, footage details, or other
sensitive information, redact it before posting.
